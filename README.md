# Audio_Converter
Place this app in **nextcloud/apps/**

# Basic information
This app is based on [Video_Converter](https://github.com/PaulLereverend/NextcloudVideo_Converter) app go and hit Star button!

## Features

* Audio Conversion
* Override or not
* Preset option


## Output supported

* MP4
* AVI
* WEBM
* M4V
* More incoming...

## Requirements

* FFmpeg
